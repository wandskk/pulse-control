# Regras de Domínio

- cada comando pertence a um único dispositivo
- um comando só pode ser executado dentro do contexto do seu dispositivo
- um dispositivo pode ter vários comandos
- a cor do botão deve vir de uma lista predefinida
- o texto SMS é configurável e representa o payload enviado
- toda execução deve gerar histórico
- falhas também devem ser registradas
